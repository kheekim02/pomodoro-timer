## ✨ Features

- **Pomodoro Timer**: Classic 25-minute work and 5-minute break intervals.
- **Audible Alerts**: Gentle chimes to notify you when a session ends.
- **Session Tracking**: Keep count of how many successful work sessions you complete in a day.
- **Interactive Priorities List**: A built-in drag-and-drop to-do list powered by `SortableJS` so you always know what to tackle next.
- **State Persistence**: Your to-dos and session counts are saved securely in your browser's local storage.
- **Glassmorphism Design**: A stunning, modern translucent UI over a deeply vibrant, animated gradient background.

## 🚀 Getting Started

No heavy build steps or package installations are required!

### Prerequisites
All you need is a modern web browser (Chrome, Firefox, Safari, Edge).

### Installation & Usage

1. **Clone the repository** (if you haven't already):
   ```bash
   git clone https://github.com/kheekim02/pomodoro-timer.git
   cd pomodoro-timer
   ```
2. **Open the App**:
   Simply open `index.html` in your favorite browser.
   ```bash
   # On macOS
   open index.html
   ```

## 🧠 How to use Focus

1. **Set your tasks**: Before starting your timer, add your daily priorities to the list. Drag them into your preferred order.
2. **Start the timer**: Click the `Start` button to begin your 25-minute work session. The background pulse will help keep you in the zone.
3. **Take a break**: When the alarm sounds, the app will automatically prompt you to take a 5-minute break. The UI will shift to a restful color palette.
4. **Repeat**: Check off tasks as you finish them, and see how many sessions you can stack!
